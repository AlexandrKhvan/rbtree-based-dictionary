# Словарь на основе Красно-Черного Дерева (Red-Black Tree Dictionary)

Этот проект представляет собой реализацию эффективного словаря на языке C. В основе хранения данных лежит сбалансированное Красно-Черное Дерево (Red-Black Tree), что обеспечивает логарифмическую сложность $O(\log N)$ для операций поиска, вставки и удаления.
